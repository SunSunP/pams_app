PAMS — Required Python packages
================================

This application has NO third-party dependencies. It is built entirely on
the Python standard library, so nothing needs to be installed with pip.

Standard library modules used (all bundled with Python 3.10+):
  - sqlite3      (database)
  - tkinter       (desktop GUI - on Linux this may need installing
                   separately via the OS package manager)
  - hashlib, hmac (password hashing / security)
  - os, re, functools, datetime  (general utilities, validation)

Evidence: `pip list` output on a freshly created virtual environment,
with no packages installed for this project:

    Package Version
    ------- -------
    pip     24.0

This confirms zero third-party packages are required to run PAMS.
